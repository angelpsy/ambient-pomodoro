# v0.1 Release Notes

**Version 0.1.0** marks the first usable release of Ambient Pomodoro.

## Features Implemented
- **Core Timer Engine**: Reliable state machine with accurate time tracking.
- **Modes**: Focus (25m), Break (5m), Long Break (15m).
- **Notifications**: "Nagging" reminders instead of jarring alarms.
- **State Persistence**: Timer restores correctly even if the app process is killed.
- **Native Android Alarms**: Uses exact alarms for reliability in background.
- **Log Viewer**: In-app tool for debugging state transitions.
- **UI/UX**: Minimal dark interface with centered controls for ergonomics.

## Technical Details
- Built with React Native & TypeScript.
- Custom Native Module for file I/O and Alarm scheduling.
- "Passive" philosophy: no stats, no strict enforcement.

## Installation
Install `app-release.apk` on any Android device.
